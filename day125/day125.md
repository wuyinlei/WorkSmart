###git命令

* git fetch origin develop:develop(拉取远程分支,必须保证这个分支已经创建好了)
* git branch(查询分支)
* git checkout develop(切换到develop分支)
* 然后各种提交命令 最后git push origin develop到远程分支,这个时候提交成功网页端会提示分支上有代码更新,是否需要合并到主分支。然后merge就行了
